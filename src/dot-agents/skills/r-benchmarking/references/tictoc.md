# tictoc package

Sources
- https://cran.r-project.org/package=tictoc
- https://raw.githubusercontent.com/jabiru/tictoc/master/README.md

Purpose
- Nested timing with tic()/toc(), with optional callbacks and stored timing history.

Notes
- Provides Stack and StackList helper classes to support nested timing.
- Best for instrumentation inside longer scripts or pipelines.
