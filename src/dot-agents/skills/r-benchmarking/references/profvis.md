# profvis package

Sources
- https://profvis.r-lib.org/
- https://raw.githubusercontent.com/rstudio/profvis/master/README.md

Purpose
- Visualizes profiling data with an interactive web UI.

Usage
- Wrap the code with profvis(expr). The result is an htmlwidget that opens in a browser when printed.

Notes
- Complements Rprof; use profvis for interactive exploration of call stacks and time hotspots.
