# Context7 slugs for R benchmarking

Resolved library IDs (Context7)
- /websites/r-project (R Project site; miscellaneous docs and conference materials)
- /rstudio/r-manuals (re-styled R manuals)
- /websites/cran_r-project_manuals (CRAN manuals index)

Notes
- Context7 does not currently expose CRAN package reference manuals for bench, microbenchmark, rbenchmark, tictoc, or profvis.
- Prefer Fetch for CRAN package pages, pkgdown sites, and GitHub READMEs.
